# React Boilerplate

This boilerplate provides a minimal setup to get React working in Vite with HMR, Bun, Biome, typescript, gitignore,
gitattributes and some biome rules.

## To update the packages

```
  bun update --latest
  pnpm update --latest
```

### This boilerplate configured

```
  biome config
  typescript config
  typescript env
  vite env
  several folders
  partials styles
```
